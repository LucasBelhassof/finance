# Termos de Uso do Finly

Versão: 2026-05-04

O Finly é uma ferramenta de organização financeira pessoal. O usuário é responsável pelas informações cadastradas, importadas e revisadas dentro da aplicação.

## Planos

- Free: recursos básicos de controle financeiro.
- Premium: recursos avançados como IA, importações avançadas, automações, insights e integrações quando disponíveis.

O acesso premium depende de assinatura ativa registrada no backend de billing. A interface pode esconder recursos, mas a autorização efetiva ocorre no servidor.

## Uso aceitável

Não é permitido tentar acessar dados de outros usuários, burlar limites do plano, explorar endpoints internos ou usar a aplicação para atividades ilegais.

## Dados financeiros

O Finly não substitui consultoria financeira, contábil, jurídica ou fiscal. Recomendações e insights são auxiliares e devem ser revisados pelo usuário.

## Mudanças

Novas versões dos Termos podem ser publicadas antes de mudanças relevantes no produto. O aceite é registrado com versão, data, IP, user agent e request id.
