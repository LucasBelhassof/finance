API administrativa

Expõe visoes gerenciais de usuarios, assinaturas, atividade, uso de IA e notificacoes administrativas. O backend valida explicitamente request.auth.user.role === "admin" antes de servir qualquer rota.

Role admin obrigatoriaAuditoria de acesso negadoMetricas operacionaisNotificacao administrativa
Capaz de fazerExpor overview, metricas financeiras agregadas, subscriptions, activity, uso de IA e criar notificacoes admin.
LimitacoesUsuario comum nao acessa; superficie depende de consultas agregadas, nao de CRUD completo de usuarios.
PermissoesBearer + role admin no backend; AdminRoute bloqueia no frontend.
Arquivos relacionadosserver/modules/admin/*, src/hooks/use-admin.ts, src/pages/admin/*.

Visao operacional

O modulo admin nao troca o escopo comum do produto; ele abre uma camada gerencial isolada para operadores autorizados. O backend ainda audita tentativa negada com evento especifico de auth.

Capaz de fazer

  • Exibir overview de usuarios, premium, churn e receita agregada.
  • Listar assinaturas/planos quando os dados existem nas tabelas agregadas.
  • Exibir atividade recente e uso de IA.
  • Criar notificacao global ou para usuarios selecionados.

Nao faz

  • Nao promove usuario a admin por API publica identificada nesta analise.
  • Nao expõe CRUD amplo de usuario final alem das metricas e notificacoes administrativas.
  • Nao permite acesso de usuario comum apenas por conhecer a rota.

Limitacoes

  • Dominios administrativos alem de overview, subscriptions, activity, AI usage e notificacoes nao foram identificados no repositorio analisado.
  • Rotas dependem de dados agregados existentes; sem dados retornam contadores zerados.

target string Sim em notificacao all ou selected Selecionados exigem userIds
title / message string Sim Tamanho e trim Validacao delegada a notificacoes

Validacoes

Auditoria de acesso negado Tentativa de usuario comum Registra evento Sem payload adicional alem do 403
Payload de notificacao POST notifications Rejeita titulo/mensagem/categoria invalidos 400

Efeitos colaterais

  • Cria notificacoes com source administrativo.
  • Ler metricas nao altera estado.
  • Tentativa negada registra auditoria de auth.

Erros esperados

401 Bearer ausente Qualquer rota Autenticar
